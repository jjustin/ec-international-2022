ifxdaq.manager

This module contains the SensorManager, which is used to operate multiple sensors simultaneously.

Classes

Class to manage multiple sensors on an abstract level.
class ifxdaq.manager.SensorManager(sensors=('ifxdaq.sensor.radar_ifx.RadarIfxAvian', 'ifxdaq.sensor.camera_cvb.CamCommonVisionBlox', 'ifxdaq.sensor.camera_irs.CamIntelRealSense'), workers=(<class 'ifxdaq.worker.RecordingWorker'>, <class 'ifxdaq.worker.VisualizationWorker'>), config_file='config.yaml')

Bases: object

Class to manage multiple sensors on an abstract level.

Currently supported sensors:

  • RadarIfxAvian

  • CamCommonVisionBlox

  • CamIntelRealSense

  • CamOpenCV

Warning

CamOpenCV has negative side effects on other cameras, e.g. CamIntelRealSense and is disabled by default.

Parameters

  • sensors (Iterable[str]) – Sensors to use. List of module names, e.g. ifxdaq.sensor.radar_ifx.RadarIfxAvian.

  • workers (Iterable[Type[WorkerABC]]) – Workers to use. List of classes, e.g. RecordingWorker.

  • config_file (Union[str, Path]) – Configuration file for the manager.
