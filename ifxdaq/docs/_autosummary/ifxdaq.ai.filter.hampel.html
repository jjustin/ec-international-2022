Bases: ifxdaq.ai.filter.abc.TrackFilterABC

Outlier removal using Hampel identifier.

The Hampel identifier is a variation of the three-sigma rule of statistics that is robust against outliers. Please have a look at https://de.mathworks.com/help/signal/ref/hampel.html for more details.

Parameters

  • time_window (float) – Window time (in seconds).

  • n_sigma (float) – Number of standard deviations.
