ifxdaq.utils.common.md5_checksum(file, chunk_size=1048576)

Calculate the MD5 checksum of a file.

Note

We know that MD5 is not state of the art and not secure. However, it is fast and we need no cryptographic hash.

Parameters

  • file (Union[str, Path]) – The file.

  • chunk_size (int) – Size of chunks read at a time.

Return type

str

Returns
