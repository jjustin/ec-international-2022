ifxdaq.sensorÔÉÅ

This package contains the base functionality and the implementations of the different sensor types.

Modules

ifxdaq.sensor.abc

This module contains frame definitions and the abstract class used as a boilerplate for all sensors.

ifxdaq.sensor.camera_cvb

This module contains the implementation to access CommonVisionBlox cameras.

ifxdaq.sensor.camera_irs

This module contains the implementation to access Intel RealSense cameras.

ifxdaq.sensor.camera_ocv

This module contains the implementation to access webcams via OpenCV.

ifxdaq.sensor.radar_ifx

This module contains the implementation to access Infineon 60 GHz radars.
