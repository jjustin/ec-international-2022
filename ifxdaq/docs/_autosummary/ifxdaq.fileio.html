This package contains implementations to write data to and read data from files.

Modules

ifxdaq.fileio.abc

This module contains the abstract classes for recorders and readers.

ifxdaq.fileio.json

This module contains the implementation of a recorder and reader for line-delimited JSON files (.json).

ifxdaq.fileio.mp4

This module contains the implementation of a recorder and reader for video files (.mp4).

ifxdaq.fileio.npy

This module contains the implementation of a recorder and reader for numpy files (.npy).

ifxdaq.fileio.sync

Tools to synchronize different data streams.
