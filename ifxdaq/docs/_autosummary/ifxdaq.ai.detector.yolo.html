class ifxdaq.ai.detector.yolo.YOLODetection(model='yolov5x_human', weights=None, threshold=0.4, nms=0.5)

Bases: ifxdaq.ai.detector.abc.DetectorABC

YOLO based object detection.

Parameters

  • model (str) –

    Model to use.

    • yolov5x_human: Custom model optimized to detect humans.

    • yolov5s: Slim & fast (big objects).

    • yolov5x: Big & more accurate. (big objects).

    • yolov5s6: Slim & fast (small objects).

    • yolov5x6: Big & more accurate. (small objects).

  • weights (Union[str, Path, None]) – Model weights for yolov5. If not provided the weights are automatically downloaded.

  • threshold (float) – Detection threshold of the model.

  • nms (float) – IoU threshold for the non-maximum suppression (nms).

Raises

RuntimeError – If model initialization failed with the provided weights.
