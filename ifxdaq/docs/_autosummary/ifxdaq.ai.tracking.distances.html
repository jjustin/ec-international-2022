ifxdaq.ai.tracking.distances.nn_cosine_distance(x, y)

Helper function for nearest neighbor distance metric (cosine).

Parameters

  • x (ndarray) – A matrix of N row-vectors (sample points).

  • y (ndarray) – A matrix of M row-vectors (query points).

Return type

ndarray

Returns

A vector of length M that contains for each entry in y the smallest cosine distance to a sample in x.
