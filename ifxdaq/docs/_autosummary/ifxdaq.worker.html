ifxdaq.worker

Implementations of workers that act as interface between the main process and background threads in other processes.

Workers are attached to a SensorProcess. From this process workers initialize background threads to allow continuous data acquisition. Workers are also an interface to communitcate between the main process and the worker threads in the SensorProcess.
