Radar data (radar.npy)ÔÉÅ

Format

Radar data is stored in radar.npy files. The NPY format is a standardized, open-source format to store numerical data.

  • file type: .npy (Numpy documentation)

  • data type: uint16 (unnormalized ADC data)

  • data shape: 4D tensor - (frames, antennas, chirps, samples)

Consequently, a tensor with the shape (4803, 3, 256, 64) would contain 4803 frames, recorded with 3 antennas, 256 chirps per frame and 64 samples per chirp. The timestamps from the related timestamps file indicate the arrival time of a radar frame on host side.

Available libraries

  • ifxdaq: ReaderNpy

  • Python: np.load

  • C: cnpy

  • Matlab: npy-matlab
