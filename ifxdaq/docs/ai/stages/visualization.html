Visualization stage

Visualize the generated labels. This will produce overview plots of the generated labels and a video based on the initial inputs with labels overlaid. Optionally, anonymization can be applied to hide persons in the resulting video.

Different anonymization styles

BLACKEN

BLUR

PIXEL

Warning

The anonymization can only be applied to detected and tracked persons. If a person was not detected, he or she will be visible in resulting video. Consequently, we do not guarantee a perfect anonymization. Please keep this in mind when sharing sensitive data with others.

Available stages

Stage name

Description & Parameters (API)

visualization
ifxdaq.ai.visualization.stage.visualization(label_dir, anonymization='NONE', anonymization_type='Box', output_name=None)

Visualize the given LABEL_DIR(s).

The labels are visualized as an overlay on the original recording (optionally with anonymization).

Parameters

  • label_dir (Sequence[Path]) – Recordings to process.

  • anonymization (str) – Anonymization to use.

  • anonymization_type (str) – Type of anonymization.

  • output_name (Optional[str]) – Optional individual name for the output directory.

Return type

List[Path]

Returns

List of folders with processed data.
